## React demo project using native math together with plotly.js

### Get started

Clone app & install the dependencies...

```bash
git clone https://github.com/dr-montasir/react-native_math-plotlyjs.git
cd react-native_math-plotlyjs
npm install
```

Start:

```bash
npm start
```
